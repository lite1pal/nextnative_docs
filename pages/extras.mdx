# Useful resources

If you need a logo, you can use [LogoFast](https://shipfa.st/tools/logo-fast)<br />
If you need web analytics, you can use [DataFast](https://datafa.st)<br />
If you need SVG illustrations, you can use [Undraw](https://undraw.co/search)<br />
If you need SVG background patterns, you can use [Hero Patterns](https://heropatterns.com/)<br />
If you need icons, you can use [Heroicons](https://heroicons.com) & [Lucide](https://lucide.dev)<br />
If you need SVG blobs, you can use [Haikei](https://app.haikei.app/)<br />
If you need Tailwind Snippets , you can use [Snippets](https://snippets.alexandru.so/)<br />
If you need Notion-like icons, you can use [Overflow](https://www.overflow.design/ni.html?ref=producthunt)<br />
If you need cute SVG doodles, you can use [Figma Doodles](https://www.figma.com/community/file/1019353050314527791)
