### Useful ressources

If you need a logo, you can use LogoFast

If you need web analytics, you can use DataFast

If you need SVG illustrations, you can use [Undraw](https://undraw.co/search)

If you need SVG background patterns, you can use [Hero Patterns](https://heropatterns.com/)

If you need icons, you can use Heroicons & Lucide

If you need SVG blobs, you can use [Haikei](https://app.haikei.app/)

If you need Tailwind Snippets , you can use [Snippets](https://snippets.alexandru.so/)

If you need Notion-like icons, you can use [Overflow](https://www.overflow.design/ni.html?ref=producthunt)

If you need cute SVG doodles, you can use [Figma Doodles](https://www.figma.com/community/file/1019353050314527791)
