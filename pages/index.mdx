# Get Started

Hey, builder! Here's a quick guide to get your project up and running!

## Start a local server

1. In your terminal, run the following commands **one-by-one**:

```shell copy
git clone https://github.com/lite1pal/lasting-habits.git [YOUR_APP_NAME]
```

```shell copy
cd [YOUR_APP_NAME]

npm i
```

> **NextNative** requires Node 18.17 or greater. Type node -v in your terminal to check version.
