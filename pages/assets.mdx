# Genereting different icon sizes from your logo

1. Once you created your logo, copy & paste it to **/assets** folder.

```
assets/
├── icon-only.png
├── icon-foreground.png
├── icon-background.png
├── splash.png
└── splash-dark.png
```

2. Run a command below to create a folder **/icons** with different icon sizes.

```shell copy
npx capacitor-assets generate
```
